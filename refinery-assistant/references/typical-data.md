# 典型数据参考 (Typical Data Reference)

> 本文件提供炼厂常见装置收率、典型组分性质范围、参考价格区间等数据。
> **所有数据仅为参考范围，实际值因原油品种、装置工况、催化剂活性等因素而异。**

## 一、典型加工路线收率

### 1.1 燃料型炼厂（原油 → 汽油 + 柴油 + 航煤）

| 装置 | 进料 | 主要产品收率 | 备注 |
|------|------|-------------|------|
| 常减压 (CDU) | 原油 | 石脑油 8-15%, 柴油 20-35%, 蜡油 25-40%, 渣油 15-30% | 取决于原油API度 |
| 催化裂化 (FCC) | 蜡油/渣油 | 汽油 40-50%, LPG 10-20%, 柴油 15-25%, 油浆 5-10% | 取决于催化剂/操作条件 |
| 连续重整 (CCR) | 石脑油 | 重整汽油(芳烃) 80-95%, H₂ 2-5% | RON 提升约 30-40 单位 |
| 加氢裂化 (HC) | VGO/DAO | 石脑油 10-25%, 柴油 40-65%, 尾油 15-30% | 灵活性高，可调节 |
| 柴油加氢 (HT) | 柴油+LCO | 精制柴油 96-99% | 脱硫率>98% |
| 烷基化 | C4 烯烃 | 烷基化油 90-95% | 产物高辛烷值低蒸气压 |
| 延迟焦化 | 减压渣油 | 焦化汽油 10-15%, 焦化柴油 25-35%, 焦蜡 20-30%, 焦炭 15-25% | 原料适应性最强 |

## 四、常用换算系数

| 换算 | 系数 | 说明 |
|------|------|------|
| 1 吨原油 ≈ 桶 | 7.2-7.6 | 取决于 API 度 (API 越高桶数越大) |
| 1 吨汽油 ≈ 升 | 1330-1380 | 密度越低升数越多 |
| 1 吨柴油 ≈ 升 | 1160-1200 | 密度越低升数越多 |
| 1 吨 MTBE ≈ 升 | 1300-1350 | — |
| 美元/桶 → 元/吨 (原油) | × 汇率 × 7.35 | 近似 1$/bbl ≈ 52-56 元/吨 |
| 布伦特 → 人民币炼厂采购成本 | +运费+保费+贴水+关税+增值税 | 通常为 Brent + 2-6 $/bbl 贴水 + 13% VAT |
