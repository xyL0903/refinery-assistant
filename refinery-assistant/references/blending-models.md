# 调合模型与计算公式参考 (Blending Models Reference)

## 一、基础原则

调合模型的核心是：**将各组分油按体积比（或质量比）混合后，预测最终产品的各项质量指标**。不同物性的混合行为不同：

| 物性 | 混合行为 | 模型 |
|------|---------|------|
| 密度、硫含量、芳烃、烯烃、苯 | 近似线性 | 体积加权平均 |
| RON、MON | 非线性（尤其有高烯烃组分时） | 线性近似 + 非线性修正项 |
| 十六烷值 | 近似线性（柴油调合） | 体积加权 |
| RVP (雷德蒸气压) | 强非线性 | 使用 Blending Index 或 Chebyshev 关联式 |
| 馏程 (T50/T90) | 非线性 | 经验关联或模型预测 |

---

## 二、辛烷值调和模型

### 2.1 线性体积模型（简化，适用于初步估算）

```
RON_blend = Σ (Vi × RONi) / V_total
MON_blend = Σ (Vi × MONi) / V_total
```

其中 Vi 为组分 i 的体积，RONi 为组分 i 的辛烷值。

**局限性**：当组分中含高烯烃催化汽油 (>35% 烯烃) 或含乙醇时，线性模型的预测值会显著偏离实际（通常高估 0.3-1.5 个 RON）。此时应注明"线性近似，实际需要 +0.5 ~ -1.0 的经验修正"。

### 2.2 乙醇效应修正（当乙醇添加 >5% 时）

```
RON_blend = 线性预测值 - ΔRON_ethanol
ΔRON_ethanol ≈ 0.2 × (EtOH_vol% / 10)   （经验修正，RVP 同步上升）
```

### 2.3 非线性经验修正（高烯烃 FCC 汽油）

当催化汽油比例 >40% 且其烯烃含量 >35% 时：

```
RON_actual ≈ RON_linear - 0.3 × (FCC_ratio - 0.4) × (FCC_olefin - 35) / 10
```

## 三、RVP 调和模型

### 3.1 Blending Index 方法

RVP 不满足体积线性，需要使用 RVP Blending Index (BI)：

```
BI_i = (RVP_i)^1.25
BI_blend = Σ (Vi × BI_i) / V_total
RVP_blend = (BI_blend)^(1/1.25)
```

**适用性**：对汽油组分间的 RVP 预测精度较好，误差通常在 ±3 kPa 以内。

### 3.2 简化线性模型（当组分性质相近时）

如果各组分的 RVP 差异不大（<30 kPa），可近似使用体积加权，但需注明误差风险。

## 四、十六烷值调和

### 4.1 体积加权（柴油调合主流做法）

```
CN_blend = Σ (Vi × CNi) / V_total
```

适用性：直馏柴油、加氢柴油、加氢裂化柴油之间的调合效果好。

### 4.2 十六烷指数 (CI) 换算

当没有实测 CN 值时，可用密度和馏程计算十六烷指数（ASTM D4737 / GB/T 11139）：

```
CI = 45.2 + 0.0892×T10 + (0.131 + 0.901×B)×T50 + (0.0523 - 0.420×B)×T90 + 0.00049×(T10² - T90²) + 107×B + 60×B²
```
其中：
- T10, T50, T90 为 10%/50%/90% 馏出温度 (℃)
- B = exp(-3.5×D) - 1，D = 15℃密度 (g/mL)

> CN 和 CI 的关系：CN ≈ CI (对于无十六烷改进剂的柴油)，差值通常在 ±2 以内。

### 4.3 催化柴油 (LCO) 的低十六烷问题

LCO 十六烷值通常仅 20-30，多环芳烃含量高 (>50%)。调和时注意：
- LCO 比例每增加 10%，CN 下降约 2-4 个单位
- 建议 LCO 比例控制在 15% 以下以保证 CN≥51
- LCO 比例过高时需加氢脱芳烃 (HDA) 处理或添加十六烷改进剂

## 五、馏程调和

### 5.1 简单混合（用于快速估算）

```
T_p,blend = Σ (Wi × T_p,i) / W_total
```

其中 T_p 为某馏出百分比的温度（T10/T50/T90），Wi 为各组分在馏程曲线上 p% 处的馏出量。

**局限**：不同组分的馏程曲线形状差异大时误差显著。

### 5.2 ASTM D86 调合 — 改进方法

更准确的方案是：先按体积比例调合各组分在每个温度点的**累计馏出体积百分比**，再反插得到 T10/T50/T90。这种方法比简单质量加权准确，但需要各组分的完整馏程曲线。

## 六、其他指标

### 6.1 密度

体积加权法（精确）：

```
Density_blend = Σ (Vi × Density_i) / V_total
```

### 6.2 硫含量

体积加权法（适用性好）：

```
S_blend = Σ (Vi × S_i) / V_total
```

### 6.3 烯烃/芳烃/苯

体积加权法：

```
Olefin_blend = Σ (Vi × Olefin_i) / V_total
Aromatics_blend = Σ (Vi × Aromatics_i) / V_total
Benzene_blend = Σ (Vi × Benzene_i) / V_total
```

> **注意**：苯含量在调合时需特别关注，因为即使各组分苯含量都 ≤1%，但多个高苯组分混合可能使成品接近上限。

### 6.4 氧含量

```
Oxygen_blend = Σ (Wi × Oxygen_i) / W_total     （质量加权，因为氧含量以 wt% 计）
```

## 七、成本优化模型 (LP)

### 7.1 标准线性规划形式

```
目标函数:  Min  Σ (Pi × X_i)
约束条件:
  Σ (A_ij × X_i) ≥ b_j    (质量指标下限)
  Σ (A_ij × X_i) ≤ b_j    (质量指标上限)
  Σ X_i = 1                (总体积归一化)
  0 ≤ X_i ≤ avail_i        (库存约束/用量上限)
```

其中：
- X_i = 组分 i 的体积比例（决策变量）
- Pi = 组分 i 的单价（元/吨）
- A_ij = 组分 i 在质量指标 j 上的值
- b_j = 成品对质量指标 j 的规格要求

### 7.2 两阶段调和建议

当被问到"成本最优方案"时，建议先做**一次无约束优化**看理论最优比例，再考虑库存约束做**第二次约束优化**，让用户看到约束造成的成本增量。

---

## 八、典型调合利润计算

```
利润 (元/吨) = 成品油出厂价 - Σ (各组分单价 × 比例) - 调合操作费用
```

调合操作费用通常为 10-30 元/吨（含电耗、添加剂、管道损耗等）。

---

> **重要**：以上公式中，线性模型（RON、十六烷值）在组分性质差异大时可能偏离实际。实际生产中应参考化验室调合小试数据对模型系数进行标定。无论如何，在本 Skill 中使用时注明"基于理论模型，实际需小样验证"。
