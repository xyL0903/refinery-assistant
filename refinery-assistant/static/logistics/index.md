# 物流配送与销售策略 (Logistics & Sales)

## 概述

成品油从炼厂到终端消费者的流通环节管理，包括出厂节奏、运输方式选择、区域价差套利和销售策略优化。

---

## 一、关键分析维度

| 维度 | 说明 |
|------|------|
| 批直 vs 零售 | 大批量直供批发 vs 加油站零售的价差 |
| 区域价差 | 不同消费地之间的价格差异（如华东 vs 华南 vs 西南） |
| 运输方式 | 管道/铁路/水路/公路的成本对比 |
| 流向管控 | 各地分公司的资源分配和流向合规 |
| 季节性波动 | 柴油/汽油消费的季节性规律 |

### 典型运输成本参考

| 方式 | 成本 (元/吨·公里) | 适合场景 |
|------|-----------------|---------|
| 管道 | 0.03-0.08 | 大批量、固定线路 |
| 水路 | 0.02-0.06 | 沿海/沿江炼厂 |
| 铁路 | 0.08-0.15 | 内陆、中长途 |
| 公路 | 0.30-0.60 | 短途、配送终端 |

## 二、典型场景

### 场景 A: "当前批直和零售哪个更划算？"
1. 对比批直价与零售指导价的差异
2. 考虑零售的运营成本（加油站在租/人工/水电等）
3. 按批量测算综合利润对比

### 场景 B: "往哪个区域出货利润最高？"
1. 列出各消费地当前价格
2. 减去对应运费
3. 净回值最高者推荐出货方向

---

## 三、跨域联动

- **← inventory**: 高库存触发加速出厂
- **→ inventory**: 销售计划影响库存目标设定
- **← risk**: 裂解价差趋势影响销售节奏
