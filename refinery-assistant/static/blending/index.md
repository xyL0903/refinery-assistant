# 成品油调和优化 (Blending Optimization)

## 概述

成品油调和是将炼厂各二次加工装置产出的组分油按比例混合，
使最终产品满足特定质量标准（国Ⅵ/出口）且成本最低的过程。

---

## 一、需要用户提供的关键参数

在开始调和计算前，收集以下信息。用户未提供时必须追问：

| 参数 | 说明 | 是否必填 |
|------|------|---------|
| 目标产品 | 如 92#/95# 汽油、0# 柴油、出口规格等 | ✅ 必须 |
| 组分油种类及性质表 | 各组分 RON/MON/密度/硫/烯烃/芳烃/苯/RVP/馏程 | ✅ 必须（可参考 typical-data.md 但需标注估算） |
| 目标产量 | 需要生产多少吨/立方米 | 建议提供 |
| 各组分可用库存量 | 跨域 inventory | 建议提供 |
| 各组分价格 | 内部结算价或采购价 | 建议提供 |
| 成本目标 | 如果有成本上限 | 可选 |
| 特殊约束 | 如某组分必须用完、某组分不能超过 x% | 可选 |

### 参数检查表

- [ ] 是否明确了目标产品标准版本（国ⅥA/ⅥB/出口）？
- [ ] 组分油性质是否完整（至少 RON + 密度 + 硫 + 烯烃，柴油需十六烷值）?
- [ ] 如果用户只给了部分数据，标注"以下计算基于部分估算数据"

## 二、调合计算流程

### Step 1: 标准选择
- 从 china-specs.md 加载对应标准限值
- 确认用户用的标准版本

### Step 2: 物性一致性检查
检查组分数据的合理性（参考 typical-data.md 的典型范围）：
- 催化汽油 RON 如果在 80-95 之外 → 提示可能录入错误
- 组分密度异常值（如重整汽油 >820 kg/m³ 或 <730）→ 提醒用户确认
- 各组分物性不全时，部分指标可用 typical-data.md 估算并标注

### Step 3: 建模与求解

**基础方法（手动/口算）**：
- 列出各组分比例 x_i
- 按线性/非线性模型计算各质量指标的预测值
- 检查是否满足标准

**LP 优化方法（推荐，让 Claude 做）**：
- 目标：最小化成本 Σ(P_i × x_i)
- 约束：各质量指标 ≤（或 ≥）规格限值
- 决策变量：各组分体积比例 x_i（总和 = 1 或 100%）
- 输出最优配方 + 预测质量

### Step 4: 库存约束检查（跨域 inventory）
- 如果用户提供了库存量，按本节第三节联动逻辑执行

### Step 5: 输出

## 三、常见调合场景处理

### 场景 A: "帮我做一个最优调合方案"
1. 收集全参数（组分表、目标、库存、价格）
2. 做 LP 优化
3. 输出方案表 + 质量预测 + 成本 + 约束检查

### 场景 B: "RON 不够，怎么调"
1. 看当前方案和目标 RON 差距
2. 列出可提高 RON 的方案（加重整汽油、MTBE、烷基化油）
3. 计算每种方案需要的加量和成本影响
4. 推荐最优的一种

**典型应对策略表**：
| 问题 | 常规方案 | 性价比推荐 |
|------|---------|-----------|
| RON 偏低 | +重整汽油 / +MTBE / +烷基化油 | MTBE 增量（RON 提升效果最大） |
| 烯烃超标 | -FCC 汽油 / +重整汽油 / +烷基化油 | 减少 FCC 汽油 + MTBE 补 RON |
| 硫超标 | -FCC 汽油 / +低硫组分 / +加氢处理 | 加氢后使用或限 FCC 比例 |
| 芳烃偏高 | -重整汽油 / +异构化油 / +MTBE | 异构化油替代重整汽油 |
| RVP 偏高 | -异构化油 / -丁烷 / -MTBE | 减少轻组分比例 |

### 场景 C: "给我一个调合方案，成本最低"
1. LP 优化求解
2. 标注哪些约束是"紧约束"（active constraint）— 这些决定了成本
3. 提供"如果放宽某个指标 × 单位，成本可降 × 元"的敏感性分析

### 场景 D: "现有配方，帮我验证一下"
1. 按用户提供的比例计算各质量指标
2. 与标准逐项对比
3. 标记通过/不通过 + 如不通过给出调整建议

### 场景 E: "低十六烷值的 LCO 怎么处理"
1. 不建议直接用 LCO 调柴油（十六烷值太低、多环芳烃太高）
2. 建议: 加氢处理后（脱芳烃）再调和，或作为燃料油组分
3. 如果必须用: 建议 LCO ≤ 10% + 配加氢裂化柴油 + 加十六烷改进剂

## 四、特殊注意事项

1. **乙醇汽油 (E10)**：添加 10% 乙醇后 RON 约提升 1-2 单位，但 RVP 显著上升（+8-15 kPa）。乙醇的辛烷值调和效应是非线性的。

2. **出口规格的限制**：
   - Euro Ⅵ 汽油要求 RON ≥ 95，比国Ⅵ 92# 要求高
   - 出口柴油十六烷值 ≥ 51 与国Ⅵ 一致
   - 但出口柴油 FAME 含量 ≤ 7%

3. **非线性效应提醒**：所有采用线性模型的调合计算必须附注"基于线性近似，实际需小样验证"。

4. **调合头在线 vs 罐式调合**：
   - 在线调合：实时调整比例，精度更高
   - 罐式调合：批次式，需考虑管线残留置换
   - 本 Skill 默认计算均为罐式调合的理论比例

5. **添加剂计算**：十六烷改进剂（2-EHN）、抗爆剂（MMT）等可以在基础调合后添加来修正，但需明确添加量对成本的影响。
