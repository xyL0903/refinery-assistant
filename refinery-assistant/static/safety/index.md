# 安全环保与碳排放 (Safety & Compliance)

## 概述

炼厂安全环保管理涵盖排污许可、VOCs 治理、废水/废气/固废处理和碳排放权管理等。
环保合规已成为制约炼厂运行的刚性约束。

---

## 一、关键环保指标

| 要素 | 主要污染物 | 排放限值（国标） |
|------|-----------|----------------|
| 废水 | COD、氨氮、石油类、硫化物 | GB 31570-2015 / GB 31571-2015 |
| 废气 | SO₂、NOx、颗粒物 | GB 31570-2015 大气污染物排放限值 |
| VOCs | 非甲烷总烃 (NMHC) | ≤60 mg/m³（有组织），泄漏检测与修复(LDAR) |
| 固废 | 废催化剂、油泥、废碱液 | 定期清运 + 危废转移联单 |
| 碳排放 | CO₂ (范围一 + 范围二) | 全国碳市场配额管理 |

---

## 二、碳市场基础

| 概念 | 说明 |
|------|------|
| 全国碳市场 | 2021年启动，覆盖发电行业，石化行业预计 2025-2026 年纳入 |
| 碳配额 (CEA) | 免费分配为主 + 部分拍卖 |
| 核证减排量 (CCER) | 可抵消 5% 的配额缺口 |
| 碳交易价格 | 2024-2026 约 70-120 元/吨 CO₂ |

### 炼厂典型碳排放源

| 来源 | 占比 | 说明 |
|------|------|------|
| 工艺加热炉/锅炉 | 40-55% | 燃料燃烧排放 |
| 催化裂化再生器 | 15-25% | 催化剂结焦燃烧 |
| 制氢装置 | 5-10% | 天然气制氢的工艺排放 |
| 火炬/放空 | 2-5% | 异常放空和处理尾气 |
| 电力消耗 | 10-20% | 范围二排放（外购电） |

## 三、典型场景

### 场景 A: "环保合规检查"
1. 检查各排污口监测数据是否达标
2. 确认排污许可证的有效期和许可量
3. 提示即将到期的证照/年检

### 场景 B: "碳配额评估"
1. 根据历史产量/能耗估算碳排放量
2. 对比碳配额分配量
3. 预测配额缺口或盈余
4. 建议交易策略
