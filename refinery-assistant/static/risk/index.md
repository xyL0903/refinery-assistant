# 价格风险管理 (Price Risk Management)

## 概述

炼厂面临的商品价格风险主要包括：
- **原油价格上涨** → 加工成本上升，采购风险
- **成品油价格下跌** → 销售收入减少，销售风险
- **裂解价差收窄** → 加工利润压缩
- **基差变动** → 期货/现货价格走势不一致

---

## 一、核心概念

| 概念 | 说明 |
|------|------|
| 裂解价差 (Crack Spread) | 成品油价格 - 原油价格（加工利润的核心指标） |
| 基差 (Basis) | 现货价格 - 期货价格 |
| 月差 (Time Spread) | 近月合约 - 远月合约价格差 |
| Backwardation | 近高远低 → 现货溢价，利好持有现货 |
| Contango | 近低远高 → 远期溢价，利好存储 |

### 裂解价差计算

```markdown
汽油裂解 = 92# 出厂价 - Brent 到厂价 × 0.85
柴油裂解 = 0# 出厂价 - Brent 到厂价 × 0.90
综合裂解 = 汽油裂解 × 汽油收率 + 柴油裂解 × 柴油收率 + 其他产品贡献

注: 0.85/0.90 是 1 吨原油产 X 吨成品油的折算系数
```

## 二、套期保值策略

| 风险暴露 | 套保操作 | 目的 |
|---------|---------|------|
| 原油采购（看涨风险） | 买入原油期货/看涨期权 | 锁定最高采购成本 |
| 成品油销售（看跌风险） | 卖出成品油期货/看跌期权 | 锁定最低售价 |
| 加工利润（裂解收窄） | 买入原油期货 + 卖出成品油期货 | 锁定加工利润 |
| 库存价值（跌价风险） | 卖出对应品种期货 | 保护库存价值 |

## 三、典型场景

### 场景 A: "当前裂解价差分析"
1. 计算当前汽油/柴油裂解价差
2. 对比历史区间（近3年百分位）
3. 给出判断：是否有利于加工？

### 场景 B: "建议如何套保"
1. 识别当前主要风险（油价上涨/下跌/裂解收窄）
2. 推荐对应的套保工具
3. 建议头寸比例（套保比例通常 50-80%）

## 四、跨域联动

- **→ blending**: 裂解价差高时建议增产、调高辛烷值多出货
- **→ planning**: 价差趋势影响装置负荷决策
- **→ crude**: 月差和贴水影响采购节奏
