# 库存管理 (Inventory Management)

## 概述

炼厂库存管理涉及原油、中间物料和成品油的罐存监控、库容调度和周转优化。
核心目标是：**在保证生产连续性的前提下，最小化库存占用成本**。

---

## 一、关键参数与指标

| 指标 | 说明 | 参考范围 |
|------|------|---------|
| 罐容 (m³) | 每个储罐的设计容量 | — |
| 安全罐存量 | 为维护生产连续性的最低库存 | 通常为 2-5 天加工量 |
| 周转次数 | 年出库量 / 平均库存 | 原油 12-20 次、成品油 10-24 次 |
| GTC (罐底) | 不可泵出的罐底油量 | 占罐容 1-3% |
| 库存天数 | (当前库存 / 日均用量) | 原油 7-15 天、成品 3-7 天 |

---

## 二、典型场景

### 场景 A: "当前 XX 库存是否合理？"
1. 获取当前库量和日均加工/销售量
2. 计算可用天数
3. 对照安全库存阈值给出判断

### 场景 B: "库容预警分析"
1. 列出所有处于高/低库存预警的罐
2. 高库存 → 建议降负荷/加大出厂/移库
3. 低库存 → 建议提负荷/减少出厂/外采

### 场景 C: "库存周转率分析"
1. 计算指定周期的周转次数
2. 对比行业基准
3. 如果偏低 → 分析原因（生产计划不合理/出厂受阻/罐容配置问题）

---

## 三、跨域联动

- **→ blending**: 组分可用库存约束调合配方中的各组分上限
- **→ planning**: 中间物料库存水平影响排产的灵活性
- **→ logistics**: 成品库存高时加速出厂，低时控量、提价
