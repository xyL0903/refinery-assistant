# 生产计划与调度 (Planning & Scheduling)

## 概述

生产计划与调度是炼厂运营的核心，根据原油采购计划、装置能力、库存和需求预测，
制定各装置负荷和物料平衡方案，实现整体效益最大化。

---

## 一、关键输入参数

| 参数 | 说明 | 是否必填 |
|------|------|---------|
| 计划周期 | 月度/周度/日度 | ✅ 必须 |
| 原油品种及数量 | 加工哪些原油、各多少吨 | ✅ 必须 |
| 各装置加工能力上限 | 常减压/FCC/重整/加氢裂化/加氢等 | ✅ 必须 |
| 各装置当前运行状态 | 是否在检修、负荷率限制 | ✅ 必须（跨域 maintenance） |
| 产品需求预测 | 汽油/柴油/航煤/石脑油等各多少 | ✅ 必须 |
| 中间物料罐存 | MTBE/烷基化油/LCO 等中间品库存 | 建议提供（跨域 inventory） |
| 产品库存目标 | 期末目标库存水平 | 可选 |
| 价格/利润预测 | 各产品预期毛利 | 建议提供 |

## 二、排产流程

### Step 1: 装置能力校核
- 检查各装置的最大/最小处理量
- 标注检修期内的装置不可用
- 计算有效生产天数

### Step 2: 物料平衡计算
```
原油加工量 → CDU 切割 → 各馏分产量
  ├── 石脑油 → 重整 (重整汽油 + 芳烃) / 化工原料
  ├── 柴油馏分 → 加氢 / 催化 / 直调
  ├── VGO → FCC / 加氢裂化 / 出蜡油
  └── 渣油 → 焦化 / 作为燃料油 / 出沥青
```

物料平衡计算原则：
1. 收率基于原油评价数据（跨域 crude）
2. 装置收率使用设计值或近期平均值
3. 注意循环物流的处理（如 FCC 回炼油浆）

### Step 3: 供需缺口分析
```
各产品总产量 = Σ (各装置对该产品的贡献)
各产品计划产量 vs 需求 → 缺口/盈余
盈余 → 可外销 / 增加库存
缺口 → 外采 / 调整装置负荷 / 改变原油品种
```

### Step 4: 方案比较

对多个候选方案做比较时，格式化为对比表：

| 方案 | 原油 | 装置负荷 | 汽油产量 | 柴油产量 | 毛利 | 优劣势 |
|------|------|---------|---------|---------|------|-------|
| A | ESPO 100% | FCC 85% | x | y | z | ... |
| B | ESPO 70%+萨中 30% | FCC 90% | ... | ... | ... | ... |

## 三、典型场景

### 场景 A: "做一个下个月的排产计划"
1. 收集原油到货计划、装置可用性、需求预测
2. 计算物料平衡
3. 输出月度排产方案 + 各装置负荷率 + 产品产量表

**输出格式**：

```markdown
## 月度排产方案 (YYYY年MM月)

### 原油加工计划
| 品种 | 加工量(吨) | 占比 |
|------|-----------|------|

### 装置负荷计划
| 装置 | 加工量(吨) | 负荷率 | 备注 |
|------|-----------|--------|------|
| CDU | ... | ...% | ... |
| FCC | ... | ...% | ... |

### 产品产量预测
| 产品 | 产量(吨) | 环比变化 | 需求满足率 |
|------|---------|---------|-----------|
| 92# 汽油 | ... | ... | ...% |
| 0# 柴油 | ... | ... | ...% |
```

### 场景 B: "装置检修，需要怎么调整排产"
1. 标记受影响的装置和停产时间段
2. 计算停工期间的产量损失
3. 提供补偿方案：
   - 提其他装置负荷
   - 外购中间组分
   - 提前备货/延迟排产
   - 改变原油品种改善收率

### 场景 C: "切换原油品种对产率的影响"
1. 对比新旧原油的 TBP 收率数据（跨域 crude）
2. 重新模拟物料平衡
3. 输出各产品产量变化 + 经济效益对比

### 场景 D: "柴油增产方案"
1. 分析当前柴油收率低的原因
2. 建议：
   - CDU 降低切割温度（多出柴油馏分）
   - FCC 提高柴油模式操作
   - 加氢裂化多产柴油模式
   - 改变原油品种

## 四、跨域联动

- **crude**: 原油品种切换时需加载原油评价数据
- **maintenance**: 计划必须考虑检修时间表
- **inventory**: 中间物料罐存影响装置的进料可用性
- **blending**: 各组分产量是调和优化的进料约束
- **risk**: 裂解价差高时考虑增产 / 低时考虑降负荷或外采

---

## 五、注意事项

1. 排产方案必须考虑装置开停工时间（通常切换需要 2-7 天过渡期）
2. 装置负荷率通常不建议超过 105%（设计值上限）
3. 物料平衡要闭合（总进料 = 总出料 + 损耗，损耗约 0.5-1.5%）
4. 不要给出超过安全规范的装置操作参数（温度/压力/空速）
